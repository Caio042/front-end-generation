Olá mundão

Me chamo Caio, tenho 24 anos e além de bibliotecário sou um desenvolvedor.

Durante minha formação em biblioteconomia tive os primeiros contatos com lógica formal e também com estruturas de organização utilizando classes que se relacionam entre si, muito parecidos com banco de dados relacionais.

Quis conhecer melhor esses assuntos e comecei minha transição para a área de programação.

Curso atualmente técnico em desenvolvimento de sistema pela ETEC Albert Einstein. Me interessei e comecei a estudar a linguagem de programação Java.

Conheci o BootCamp da generation em desenvolvimento web Java FullStack, me apaixonei pelo projeto e ingressei no programa.

Aprendo hoje na generation conhecimentos técnicos de programação, mas além disso aprendo também mentalidades e habilidades comportamentais que são essenciais na atuação profissional como comunicação, persistência e trabalho em equipe.

Sou uma pessoa que se interessa muito em aprender cada dia mais, e para mim, adquirir conhecimento é um ato permanente.

Eu gosto de novos desafios e enxergo neles uma oportunidade de aprender e melhorar como pessoa.

Sou responsável e não gosto de deixar a minha equipe na mão, faço tudo o que for possível para que consigamos atingir os objetivos juntos.

Se eu fosse me definir profissionalmente, não seria como desenvolvedor java. Mas simplesmente desenvolvedor. Eu desenvolvo soluções, e para mim, java é a ferramenta que uso. Caso o java deixe de ser a ferramenta ideal, estudarei novas tecnologias e aplicarei ao desenvolvimento.

Posso me adaptar facilmente, e como vocês podem ver, já estou preparado para fazer isso futuramente.
